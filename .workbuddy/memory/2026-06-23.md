## Bug 修复 — 云开发未初始化

### 问题现象
小程序运行报错 `Cloud API isn't enabled, please call wx.cloud.init first`，导致登录、加载题库、AI 解析全部失败。

### 根因
`src/App.vue` 的 `onLaunch` 中直接调用 `userStore.doLogin()` → `login()` → `uni.cloud.callFunction()`，但从未调用 `uni.cloud.init()` 初始化云开发 SDK。

### 修复内容
1. **App.vue**：在 `onLaunch` 最前面添加 `uni.cloud.init({ traceUser: true })`，带 try-catch 错误处理
2. **src/project.config.json**：新建，添加 `cloudfunctionRoot: "cloudfunctions/"`，让微信开发者工具能识别云函数目录进行部署
3. **dist project.config.json**：同步添加 `cloudfunctionRoot`
