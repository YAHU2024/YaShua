## 完成：修复 aiParse 云函数 response_format 与 Prompt 格式冲突 Bug

日志显示 AI 解析报"未能从文本中解析出题目"。根因：
- `aiParse/index.js:73` 设置了 `response_format: { type: 'json_object' }`，强制 AI 输出 `{...}`
- 但 `buildPrompt()` 要求 AI 输出 `[...]` 纯数组
- 格式冲突导致 DeepSeek 输出不稳定，parseAIResponse 返回空数组

### 修复内容（cloudfunctions/aiParse/index.js）
1. **Prompt 格式修正**：输出从 `纯 JSON 数组` → `纯 JSON 对象 {"questions": [...]}`
2. **System prompt 修正**：从"输出 JSON 数组" → "输出 JSON 对象格式 {"questions": [...]}"
3. **parseAIResponse 兼容扩展**：支持 `result`/`items`/`list` 等更多包装键名
4. **错误消息优化**：空结果提示更具体 + 文本预览日志

构建验证通过。需重新部署 aiParse 云函数后生效。
